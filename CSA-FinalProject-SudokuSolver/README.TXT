PROJECT TITLE: CSA Sudoku Solver
PURPOSE OF PROJECT: Final project for APCSA
VERSION or DATE: 5/31/2024
AUTHORS: Yunosuke Nakamura and Leonard Jin
USER INSTRUCTIONS:
First, run DataFlair-Sudoku-Solver.py. May need to alter the file path of the image.
The Python code converts the image of Sudoku by using vision processing and an MNIST number classifier.
The converted sudoku's information is stored in readData.txt.
Then run main.java, which will display GUI, and click "solve" to solve sudoku using backtracking.
DEPENDENCIES:
need to run 
pip install opencv-python
to install opencv on the enviroment
